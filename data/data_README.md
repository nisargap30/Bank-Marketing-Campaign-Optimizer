# Dataset

The dataset used in this project is the **Bank Marketing Dataset** from the UCI Machine Learning Repository.

**Direct link:** https://archive.ics.uci.edu/dataset/222/bank+marketing

## Download Instructions

1. Visit the link above and click **Download**
2. Unzip the archive — you'll see several CSV files
3. Use **`bank-additional-full.csv`** (the full dataset, semicolon-separated, 41,188 rows)
4. Place it in this `data/` folder if running locally

## Why it's not included here

UCI's terms request that the dataset not be redistributed. Please download it directly from the source above.

## Citation

Moro, S., Rita, P., & Cortez, P. (2012). Bank Marketing Dataset. UCI Machine Learning Repository. https://doi.org/10.24432/C5K306
